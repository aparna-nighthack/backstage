
## Overview

Welcome to the Nighthack project! This documentation provides information about the project's setup, usage, and functionality. Nighthack is a fully responsive web application designed with modern design principles, utilizing Bootstrap 5 for styling and layout. The project includes various features showcased on the homepage and is structured to be easily navigable and extensible.

## Rebranding

The Nighthack application is a rebranded version of the original Backstage project. All occurrences of "Backstage" have been updated to "Nighthack" across the entire codebase, including the user interface text, documentation, metadata files, and configuration settings. This ensures brand consistency throughout the project.

## Getting Started

To set up the Nighthack project, follow these instructions:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/nighthack.git
   ```
   
2. **Navigate to the project directory:**
   ```bash
   cd nighthack
   ```

3. **Open `index.html` in your preferred web browser.**

## Project Structure

The Nighthack project is organized into the following key files:

- **`src/index.html`:** The main HTML file, featuring a responsive layout with a navigation bar, cards, and a modern grid system.
  
- **`src/style.css`:** Contains all the styling rules, ensuring a consistent look and feel that aligns with the Nighthack brand.
  
- **`src/scripts.js`:** Includes JavaScript code for interactive functionality, such as navigation menu toggling.
  
- **`config/settings.json`:** Configuration file that holds metadata, theme settings, and API endpoints.

- **`docs/documentation.md`:** This documentation file, providing an overview and setup details.

- **`README.md`:** Contains a high-level overview and setup instructions for the entire project.

## Usage Instructions

- **Navigation:** Use the top navigation bar to access different sections of the application. The navbar is fully responsive and works well across devices.
  
- **Content Display:** The homepage uses cards to display feature information. These cards are styled and laid out using Bootstrap’s grid system.

- **Customization:** Modify the `config/settings.json` file to adjust project-specific settings, such as API endpoints or feature flags.

## Theme and Branding

The theme of the Nighthack project is designed to reflect a modern and engaging brand identity:

- **Primary Colors:** Dark-themed backgrounds with contrasting text for visibility.
- **Responsive Design:** Adjusts to various screen sizes, ensuring a great user experience on desktops, tablets, and smartphones.

## API Endpoints

- `/api/users`: Endpoint to interact with the user service.
- `/api/products`: Endpoint to interact with the product service.
